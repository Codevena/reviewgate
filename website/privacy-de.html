Seitennachweis / Datenschutz

Kein Tracking.
Klare Fakten.

Die öffentliche Reviewgate-Website ist eine statische GitHub-Pages-Seite. Sie hat weder Konten noch Formulare, Analysewerkzeuge, Werbepixel oder ein Cookie-Banner, weil diese Seite keine nicht notwendigen Cookies setzt.

Verantwortlicher

Der verantwortliche Herausgeber und seine Kontaktdaten sind im Codevena-Impressum angegeben.

Hosting-Protokolle

Diese Seite wird über GitHub Pages ausgeliefert. Beim Seitenaufruf können GitHub und dessen Auslieferungsinfrastruktur technische Anfragedaten wie IP-Adresse, Zeitpunkt, angeforderten Pfad, User-Agent und Referrer verarbeiten, um den Dienst bereitzustellen und abzusichern.

Diese Verarbeitung wird von GitHub verantwortet und in der allgemeinen Datenschutzerklärung von GitHub beschrieben. Reviewgate erhält für diese Website kein Analyse-Dashboard mit Daten einzelner Besucher.

Lokale Ressourcen

Azeret Mono und Teko werden unter der SIL Open Font License direkt von dieser Domain ausgeliefert. Beim Laden einer Seite wird Google Fonts nicht kontaktiert. Das JavaScript der Website steuert Navigation, Kopier-Schaltflächen und Darstellungseffekte; es schreibt weder Cookies noch Browser-Speicher.

Externe Links

Links zu GitHub, npm, Codevena und Anbieter-Dokumentationen verlassen diese Website. Der jeweilige Zieldienst erhält die üblichen Anfragedaten und wendet seine eigenen Datenschutzbestimmungen an.

Ihre Anfragen

Für Auskunft, Berichtigung, Löschung oder andere Datenschutzanfragen an den Website-Herausgeber verwenden Sie bitte die Kontaktdaten im Impressum. Gesetzliche Rechte und Beschwerden bei der zuständigen Aufsichtsbehörde bleiben unberührt.

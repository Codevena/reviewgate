Site record / Privacy

No tracking.
Plain facts.

Reviewgate's public website is a static GitHub Pages site. It has no account system, forms, analytics, advertising pixels or cookie banner because this site does not set non-essential cookies.

Controller

The responsible publisher and contact details are identified in the Codevena imprint.

Hosting logs

This site is delivered through GitHub Pages. When you request a page, GitHub and its delivery infrastructure may process technical request data such as your IP address, timestamp, requested path, user agent and referrer to deliver and secure the service.

That processing is controlled by GitHub and described in the GitHub General Privacy Statement. Reviewgate does not receive a visitor-level analytics dashboard from this site.

Local assets

Azeret Mono and Teko are served directly from this domain under the SIL Open Font License. Loading a page does not contact Google Fonts. The site JavaScript provides navigation, copy buttons and presentation effects; it does not write cookies or browser storage.

External links

Links to GitHub, npm, Codevena and provider documentation leave this website. The destination service receives the normal request data and applies its own privacy terms.

Your requests

For access, correction, deletion or another privacy request concerning the site publisher, use the contact information in the imprint. Statutory rights and competent supervisory-authority remedies remain unaffected.

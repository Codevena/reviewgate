# What Reviewgate is (one paragraph context)

Reviewgate is a multi-agent code-review tool that auto-invokes inside
Claude Code's agent loop. After Claude Code edits files, Claude Code's
hooks (PostToolUse + Stop) trigger Reviewgate, which spawns several
heterogeneous LLM reviewers (Codex, Gemini, Claude as a fresh
subprocess) as isolated processes, aggregates findings under a
severity-weighted veto rule, and blocks Claude's turn-end until the
findings are addressed. Reviewgate maintains a curated per-repo
"brain" of conventions and false positives that calibrates future
reviewers.
